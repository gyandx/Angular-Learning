Services:- These are simple classes that are used to the encapsulate(reuse) the same business login with different components in angular, rather than making new functions in for a same thing in every component.

Dependency Injection:- This is used to inject the service or the instance in the particular component where this service can be used, as the component depend on that service dependency to perform some tasks.

Hierarchial of services

The angular services hierarcy moves from top to down like tree not from down to up.

  • App Module is the highest provider level. The same instance to all components, directives, all other child components
  • App Component is the second provider level.The same instance is available for the component and all its child components and child of child components
  • Any other component. The same instance is available for ony that component not for any other component
