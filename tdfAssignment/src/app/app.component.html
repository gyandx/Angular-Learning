Template Driven Form Assignment

Add form with the following input and validators.

  1. Mail Address (should not be empty & should be an mail)
  2. A Dropdown which allows the user to select 3 type of subscriptions ('Basic', 'Advance, 'Pro) and set 'Advance' as default.
  3. A password field (should not be empty
  4. A submit button

Display a warning message if the form is invalid and touched and display warning message under each input filed if it is invalid.

On submitting the form print the value of the fields.
